# ✈️ Travello App

**Travello** is a modern and responsive travel booking application built with Flutter. It features a
clean, professional UI designed for an optimal user experience across different screen sizes.

---

## 🚀 Features

- **Responsive UI:** Fully optimized for different screen sizes using `flutter_screenutil`.
- **Smooth Onboarding:** Interactive onboarding screens with a progress indicator to guide new
  users.
- **Complete Authentication Flow:**
    - **Login:** Professional login interface with email and password fields.
    - **Registration:** Comprehensive registration form for new users.
    - **Verification:** Secure OTP/6-digit verification screen.
- **Home/Welcome Screen:** A welcoming landing page after successful authentication.
- **Custom Widgets:** Highly reusable components like `CustomTextField` and `CustomElevatedButton`
  for consistent styling.
- **Modern Design:** Uses high-quality assets and Google Fonts for a premium look and feel.

## 🛠️ Tech Stack

- **UI & Styling:**
    - `flutter_screenutil` (for responsiveness)
    - `smooth_page_indicator` (for onboarding dots)
    - `google_fonts` (for professional typography)
    - `flutter_verification_code` (for the OTP screen)
- **State Management:** StatefulWidget / setState

---

## 📂 Project Structure

```text
lib/
├── CustomWidgets/    # Reusable UI components (TextFields, Buttons)
├── screens/          # App screens (Splash, Onboarding, Login, Register, Verify, Welcome)
├── utils/            # App styles and color constants
└── main.dart         # Entry point of the application
```

## ⚙️ Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/AmiraAbdel-fatah/Travel-UI-App.git
   ```

2. **Navigate to the project directory:**
   ```sh
   cd Travello_App
   ```

3. **Install dependencies:**
   ```sh
   flutter pub get
   ```

4. **Run the app:**
   ```sh
   flutter run
   ```
